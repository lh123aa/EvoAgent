# EvoAgent - 让Agent学会自我进化

> Copy. Deploy. Evolve.

一个让AI Agent具备自我进化能力的开源工具包。

## 🚀 这是什么？

EvoAgent是一套Agent自进化系统配置方案。它不是让Agent"学会用工具"，而是让Agent**升级自己的操作系统**——记忆系统、检索方案、迭代机制、学习闭环。

**核心理念**：
- Agent不应该"用完即忘"
- Agent应该在每次交互中变得更强
- Agent应该能自主迭代升级

## 🌟 核心特性

| 特性 | 说明 |
|------|------|
| **四层记忆架构** | 常驻记忆 + 会话归档 + 知识库 + 历史存档 |
| **技能闭环机制** | 使用→记录→分析→迭代，越用越强 |
| **自主迭代SOP** | 发现问题→设计方案→执行→测试→备份回滚 |
| **文件检索方案** | 不依赖平台RAG，稳定可靠 |
| **Wiki知识融合** | 结构化知识管理，主动入库 |

## 🔥 为什么选择EvoAgent？

### vs Hermes Agent

| 对比项 | Hermes Agent | EvoAgent |
|--------|--------------|----------|
| 进化层面 | 技能级（SKILL.md优化） | **系统级**（整个架构升级） |
| 记忆系统 | 四层架构 | 四层架构 + 容量管理 + 存档机制 |
| 迭代方式 | DSPy+GEPA自动优化 | **自主迭代SOP**（无需额外工具） |
| 开源程度 | MIT | MIT |
| 部署难度 | 需要独立部署 | **复制配置即可使用** |

**一句话总结**：Hermes让Agent学会用工具，EvoAgent让Agent升级操作系统。

## 📖 快速开始

### 1. 阅读文档

从 [迭代SOP](./docs/迭代SOP.md) 开始，了解Agent自进化的方法论。

### 2. 使用模板

复制 \`docs/记忆模板/\` 中的模板到你的Agent配置中。

### 3. 建立闭环

按照 [技能闭环机制](./docs/技能闭环机制.md) 建立你的Agent迭代循环。

### 4. 持续进化

让Agent在每次交互中学习、记录、迭代。
